# Interactive CV (Creative/Modern)

React + Vite + Tailwind interactive CV in the style you shared:
- Creative hero header
- Skill bars
- Degrees + experience cards
- Language rings
- Drag & drop layout (Edit mode)
- Bilingual (EN/GR)
- PDF export

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this repo to GitHub
2. Import in Vercel
3. Deploy

## Where to edit content
- `src/content/en.json`
- `src/content/el.json`
